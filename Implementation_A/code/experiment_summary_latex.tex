
\section{Experimental Results - Monte Carlo Option Pricing}

\subsection{Setup}
\begin{itemize}
    \item Problem: European call option pricing via Black-Scholes Monte Carlo
    \item Payoff: $\max(S_T - K, 0)$ where $K = 100$
    \item Dataset: 1000 parameter combinations
    \item Parameters: $S_0 \in [80.0, 120.0]$,
          $\sigma \in [0.15, 0.45]$,
          $T \in [0.5, 2.0]$
    \item Monte Carlo samples: 1,000,000 per option
    \item Neural architecture: 3-64-64-1 with Tanh activations
    \item Training: 3000 epochs, Adam optimizer ($\alpha = 0.001$)
\end{itemize}

\subsection{Conclusion}
The neural surrogate model achieves $R^2 = 0.8704$ with 6642$\times$
speedup over traditional Monte Carlo simulation, demonstrating effective approximation
of European call option prices. The model provides deterministic pricing suitable for
high-frequency trading applications.
